# ivosteimanis.github.io

Personal academic website of **Ivo Steimanis**, PostDoc at [Marburg University](https://www.uni-marburg.de/en/fb02/research-groups/economics/sustuse) in the research group on Sustainable Use of Natural Resources.

## About

This site showcases my research on decision-making under climate change, combining behavioral and experimental economics with field experiments, surveys, RCTs, and meta-analyses. It includes my publications, working papers, ongoing projects, and an interactive map of research field sites.

**Live site:** [ivosteimanis.github.io](https://ivosteimanis.github.io)

## Tech Stack

- [Astro 5](https://astro.build/) with [Tailwind CSS](https://tailwindcss.com/)
- React islands for interactive components (Leaflet map)
- Deployed via GitHub Actions to GitHub Pages

## Development

```bash
cd astro-site
npm install
npm run dev       # Start dev server at localhost:4321
npm run build     # Production build to dist/
```
